## ✏️ How to update this file

After landing a new demo or renaming a persona:

1. Add or update the row under the appropriate section.
2. If you've shipped a new "latest", update the **Base URL** auto-forward note + the section heading.
3. Commit with a `links: …` message prefix so the file shows up cleanly in `git log`.
